# 🎮 Deadlock FanPage

An unofficial fan-made website inspired by *Deadlock*, a new game by Valve. This project showcases characters, mechanics, updates, and gameplay elements through a modern and interactive interface.

---

## 🧠 About the Project

The main idea of this page is to present *Deadlock* to users by exploring its core elements, including characters, gameplay mechanics, patch updates, and visual content.

Users can:

* Discover featured characters on the home page
* Watch gameplay videos and trailers
* Browse a carousel with in-game images
* Access the latest patch notes with detailed updates
* Explore characters, their abilities, and characteristics
* Learn game mechanics such as movement, deny system, and item purchasing
* Navigate between multiple sections (Home, Patches, Mechanics, Characters)

---

## ✨ Features

* 🎠 Image carousel
* 🎮 Character and mechanics selection system (dynamic content display)
* 📱 Responsive layout
* 🎥 Embedded gameplay and ability videos
* 🔗 External link to Steam page
* 🧭 Multi-page navigation system
